\subsubsection*{f}

$X \sim \mathcal{N}(-1, 4)$, so $\mu = -1$, $\sigma = 2$. Standardize: $Z = \frac{X+1}{2} \sim \mathcal{N}(0,1)$.

For $a$: $P[X \leq a] = P\!\left[Z \leq \frac{a+1}{2}\right] \leq 0.1$.
By symmetry of $\mathcal{N}(0,1)$, $P[Z \leq -z] = P[Z \geq z]$, so we need $\frac{a+1}{2} = -z$ where $P[Z \geq z] = 0.1 \Rightarrow z \approx 1.28$.
\[ \frac{a+1}{2} = -1.28 \implies a = 2(-1.28) - 1 = -3.56 \]

For $b$: $P[X \geq b] = P\!\left[Z \geq \frac{b+1}{2}\right] \leq 0.2$.
From tables, $P[Z \geq z] = 0.2 \Rightarrow z \approx 0.84$.
\[ \frac{b+1}{2} = 0.84 \implies b = 2(0.84) - 1 = 0.68 \]
